### Stack Decision Table

| Layer              | Technology                         | Justification                                                                      |
|--------------------|------------------------------------|------------------------------------------------------------------------------------|
| **Frontend**       | Flutter 3.x                        | Single codebase โ iOS, Android, Web. Best-in-class for glassmorphism & 120Hz animations |
| **State**          | Riverpod 3.0                       | Cleanest handler for "Worksheet" logic across screens                              |
| **Math Engine**    | `decimal` + `curo` (Dart)          | High-precision finance packages โ eliminates floating-point rounding errors        |
| **Local Storage**  | Isar Database                      | Lightning-fast local DB for saved calculations                                     |
| **Backend**        | FastAPI (Python)                   | Handles affiliate routing, live rate fetching, lead-gen scoring                    |
| **Web Deploy**     | PWA (manifest.json)                | "Install" to desktop/phone, offline-capable via service workers                    |
| **Design System**  | Glassmorphism / Nordic Minimalism  | Whitespace, blur effects, premium typography (Inter / SF Pro)                      |

### Why Native Dart, Not Python Bridge?

> *"Bridging Python into a mobile app for simple financial math is like hiring a semi-truck
> to deliver a single envelope."*

All calculation logic lives **client-side in Dart** for zero-latency offline operation.
FastAPI handles only server-side business logic (affiliates, live data, analytics).
